# Currency Converter (CLI)

A simple Python command-line app that converts currencies using real-time exchange rates. Built with the exchangerate-api.com API.

## Features

- Input amount and currency codes
- Real-time conversion using free API
- CLI-based interaction
- No API key required

## Technologies Used

- Python 3
- requests module
- exchangerate.host API

## Sample Output
=== Currency Converter ===
Enter amount to convert: 100
Convert from (currency code, e.g. USD): usd
Convert to (currency code, e.g. EUR): eur

100.00 USD = 89.61 EUR
